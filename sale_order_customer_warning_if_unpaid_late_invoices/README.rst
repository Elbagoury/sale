=====================================================
Show Warning if Customer has unpaid and late invoices
=====================================================

Pop up warning on Sale Order if a selected customer has one or more unpaid
invoices that are late.

Configuration
=============
No special configuration needed.

Usage
=====
Install from Apps.

Known issues / Roadmap
======================
There are no known issues with this module.
